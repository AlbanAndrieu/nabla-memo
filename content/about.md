# About Nabla Memo

This is a sample Hugo project demonstrating deployment with Vercel.

## Technology Stack

- **Hugo**: Fast and flexible static site generator
- **Vercel**: Modern deployment platform for frontend frameworks
- **GitHub Actions**: Automated CI/CD workflows
- **PaperMod**: Clean and minimal Hugo theme

## Purpose

This project serves as a template for deploying Hugo sites with Vercel and automating deployment through GitHub Actions.
